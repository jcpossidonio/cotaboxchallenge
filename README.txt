Cotabox - Challenge

Para desenvolvimento do projeto foi utilizado o Sublime Text 3
em um Servidor local Apache 2.4/ Mysql 5.7 / PHP 7.2
Utilizando Zend Framework, Bootstrap e Google Charts para montagem do gráfico do projeto
O banco de dados MySQL está no arquivo cotabox.sql na raiz do projeto.

As configurações de banco de dados ficam em: cotaboxchallenge/application/configs
As pastas de Layouts, Controllers, Models e Views ficam em: cotaboxchallenge/application
O arquivo de CSS fica em: cotaboxchallenge/css
